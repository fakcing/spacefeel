Мне нужно создать универсальный компонент плеера для проекта spacefeel на Next.js. Компонент должен поддерживать два источника: Yani/Alloha (основной) и VideoCDN (запасной).

Параметры компонента:

type: 'movie', 'tv' или 'anime'.

tmdbId: ID из базы TMDB (для фильмов и сериалов).

shikimoriId: ID из базы Shikimori (для аниме).

Функционал:

Переключатель источников: Сделай две аккуратные кнопки над плеером: "Сервер 1" и "Сервер 2".

Запрос к VideoCDN: Напиши функцию, которая формирует запрос к https://videocdn.tv/api/short.

Если type === 'movie', используй параметр tmdb_id.

Если type === 'tv', используй параметр tmdb_id.

Если type === 'anime', используй shikimori_id.

Обработка серий (для сериалов и аниме): >    - Добавь логику выбора сезона и серии.

При переключении серии оба плеера должны обновлять свои iframe.

Интерфейс: Сделай список доступных озвучек от VideoCDN (они приходят в массиве translations). Стилизуй их через Tailwind CSS как небольшие кликабельные плашки под плеером.

Отказоустойчивость: Если один плеер выдает ошибку (Timeout или 404), автоматически показывай кнопку "Попробовать запасной сервер".

Используй 'use client', так как запросы должны идти с IP пользователя для обхода блокировок Vercel.
