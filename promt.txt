Я разрабатываю проект spacefeel на Next.js (App Router) + Tailwind CSS. Мне нужно создать один мощный клиентский компонент UniversalPlayer.tsx, который решит проблемы блокировок, дублирования серий и отсутствия выбора источников.

1. Контекст и Данные:

Компонент принимает пропсы: type ('movie', 'tv', 'anime', 'cartoon'), tmdbId (для фильмов/сериалов/мультиков) и shikimoriId (для аниме).

Используй два API: Yani/Alloha (основной) и VideoCDN (запасной). Токен VideoCDN бери из process.env.NEXT_PUBLIC_VIDEOCDN_TOKEN.

2. Решение проблемы блокировок (Timeout/404):

Весь компонент должен быть 'use client'. Запросы к API должны идти прямо из браузера пользователя.

Реализуй две кнопки переключения: "Сервер 1 (Alloha)" и "Сервер 2 (VideoCDN)".

Если выбранный сервер не отвечает 7 секунд, покажи уведомление: "Похоже, сервер заблокирован в вашем регионе. Попробуйте сменить сервер или включить VPN".

3. Логика фильтрации и выбора (Fix Duplicates):

Для сериалов, аниме и мультсериалов напиши функцию группировки:

Сначала по Сезонам (season).

Внутри сезона по уникальным номерам серий (episode). Избавься от дублей, если одна серия приходит несколько раз из-за разных озвучек.

Создай три выпадающих списка (Select) или ряда кнопок:

Сезон (если их > 1).

Серия.

Озвучка/Перевод (для VideoCDN берется из массива translations).

4. Универсальность контента:

Для фильмов и полнометражных мультфильмов (movie / cartoon): показывай сразу плеер.

Для сериалов и мультсериалов (tv / cartoon сериал): используй tmdbId и логику сезонов.

Для аниме: используй shikimoriId.

5. Дизайн и Мобильность (Tailwind):

Контейнер плеера должен иметь соотношение 16:9 (используй aspect-video).

Мобильная версия: На телефонах (экран < 768px) кнопки выбора серий должны превращаться в прокручиваемый горизонтальный список, а выпадающие меню должны быть крупными для удобного нажатия пальцем.

Стилизуй всё под "Dark Space" тему: темный фон, скругленные углы (rounded-xl), активные элементы с синим или фиолетовым свечением.

6. Технические детали:

Используй useState для хранения текущего сервера, сезона, серии и озвучки.

Сохраняй выбор сервера и озвучки в localStorage, чтобы при переходе к следующему фильму настройки подтянулись автоматически.

Добавь iframe с атрибутами allowFullScreen и referrerpolicy="no-referrer-when-downgrade".
