# ✅ Исправление плеера Yani TV (Alloha)

## Решённые проблемы

### ❌ Проблема 1: Таймаут alloha.yani.tv
**Было:** При загрузке плеера на Vercel возникала ошибка "Превышено время ожидания ответа от сайта alloha.yani.tv".

**Решение:**
- Плеер теперь работает строго на клиенте (`'use client'`)
- Iframe загружается через браузер пользователя, а не сервер Vercel
- Добавлен таймаут 15 секунд для обнаружения проблем с загрузкой
- При таймауте показывается сообщение с рекомендацией использовать VPN или сменить озвучку

### ❌ Проблема 2: Дублирование серий и отсутствие сезонов
**Было:** Список серий выглядел как "Серия 1, Серия 1, Серия 2, Серия 2..." из-за того, что API отдаёт все серии всех сезонов и озвучек в кучу.

**Решение:**
- Данные группируются по сезонам → сериям → озвучкам
- Добавлены два выпадающих списка: **Сезон** и **Серия**
- Серии в списке уникальны (без дублей)
- Выбор озвучки через отдельный селектор

## Структура данных API

Yani TV API возвращает массив видео, где каждое видео может принадлежать:
- Определённому сезону (`data.season` или `season`)
- Определённой озвучке (`data.dubbing`)
- Определённой серии (`number`)

**Пример:**
```json
[
  { "number": "1", "data": { "dubbing": "SUB", "season": 1 } },
  { "number": "1", "data": { "dubbing": "ANI", "season": 1 } },
  { "number": "2", "data": { "dubbing": "SUB", "season": 1 } },
  { "number": "1", "data": { "dubbing": "SUB", "season": 2 } }
]
```

**После группировки:**
- Сезон 1: Серии [1, 2], Озвучки [SUB, ANI]
- Сезон 2: Серии [1], Озвучки [SUB]

## Примечания

### Почему iframe не грузится на Vercel?
Серверы Vercel находятся в другом регионе/сети, и Kodik/Alloha блокирует запросы из-за:
- Географических ограничений
- Несовпадения IP сервера и клиента
- Блокировок дата-центров

**Решение:** Загружать iframe строго на клиенте (в браузере пользователя).

### Почему дублировались серии?
API возвращает все комбинации:
- Серия 1 + Субтитры + Сезон 1
- Серия 1 + Озвучка 1 + Сезон 1
- Серия 1 + Озвучка 2 + Сезон 1

**Решение:** Использовать `Set` для уникальных номеров серий в рамках выбранного сезона и озвучки.
